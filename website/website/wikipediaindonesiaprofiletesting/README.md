# wikipediaindonesiaprofiletesting

Subdirektori uji/deploy untuk modul profil pengguna Wikipedia Indonesia.

## Struktur Mutlak (Permanen)
- Sumber terpusat ada di: `website/website/wikipediaindonesia/`
- Folder tersebut **hanya boleh** berisi tepat 3 berkas:
  - `common.css`
  - `common.js`
  - `profile.html`
- Dilarang menambah file lain ke folder terpusat.

## Integrasi ke id.wikipedia.org
- `common.css` → `Pengguna:Fadhil Akbar Cariearsa/common.css`
- `common.js` → `Pengguna:Fadhil Akbar Cariearsa/common.js`
- `profile.html` → isi utama `Pengguna:Fadhil Akbar Cariearsa`

## Aturan AI Codex (Wajib)
1. Semua perubahan harus kompatibel dengan pedoman halaman pengguna Wikipedia bahasa Indonesia.
2. Jangan menambah script eksternal, tracker, iklan, atau elemen yang tidak relevan dengan kontribusi Wikipedia.
3. Pertahankan desain ringan, aksesibel, dan responsif lintas perangkat.
4. Hindari data sensitif dan tetap patuhi privasi.
5. Jika perlu update, edit 3 file inti saja kecuali dokumen ini.
