## Prinsip Implementasi
- Konten halaman pengguna harus mendukung kolaborasi/profil kontribusi, bukan situs komersial.
- Gunakan HTML/CSS/JS yang aman dan minimal.
- Tidak boleh memakai library pihak ketiga.
- Komponen harus stabil di HP, tablet, laptop, desktop.

## Checklist Rilis
- Validasi sintaks JS/CSS.
- Verifikasi jam/runtime berjalan.
- Verifikasi tampilan responsif <720px dan >=720px.
- Verifikasi tidak ada file tambahan di folder `wikipediaindonesia`.
