# DreamBusiness

Folder ini menjadi root utama untuk aset dan source game **DreamBusiness**.

Struktur utama:

- `games/dreambusiness`: implementasi game business simulator yang sebelumnya berada di `src/games/dreambusiness`.

Import dari aplikasi tetap menggunakan alias yang sama:

- `@/games/dreambusiness`

Alias tersebut diarahkan ke folder ini lewat konfigurasi `mindmapmaker/tsconfig.json`.
