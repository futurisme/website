# shareideas

Folder utama terpisah untuk seluruh elemen frontend fitur `/shareideas`.

## Isi
- `ShareIdeasPage.tsx`: replikasi 100% dari konsep, logika, dan visual aset `game-ideas`.
- `head.tsx`: loader stylesheet khusus `shareideas.css`.
- `index.ts`: export terpusat.

Struktur ini menjaga seluruh elemen fitur berada di satu folder utama `shareideas`.
